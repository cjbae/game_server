# game_server
IOCP를 이용한 mmorpg game server

# Map
1. 크기 - 100x100
2. 생성 - 랜덤
3. 종류 - 이동 가능 지역, 이동 불가능 지역 2개
4. 관리 - 섹터로 나눠서 관리

# NPC
1. 구현 - Lua 스크립트 언어
2. 길찾기 - A* 알고리즘 사용
3. 종류 - 각각 다른 행동을 하는 3종류의 NPC, 랜덤으로 움직이다가 공격한 유저를 인식해서 따라간다.
  1) 비 선공, 때려야 움직임
  2) 선공, 랜덤으로 움직임
  3) 비 선공, 랜덤으로 움직임
  

# Player
1. 레벨, 공격력 경험치, 골드 등 스텟이 존재
2. 시야 구현
3. 동기화 구현
4. 채팅 구현
5. 사냥을 통해 성장 가능
6. 상점에서 스텟 업그레이드 가능
